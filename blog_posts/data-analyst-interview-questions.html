Data Analyst Interview Questions & Best Answers (2026 Edition)

The most frequently asked Data Analyst interview questions—explained in simple, job-ready answers.

Companies in 2026 are looking for Data Analysts who can work with SQL, Excel, Power BI/Tableau, and business logic. These questions are asked in 90% of interviews and will help you prepare quickly.

---

1. What does a Data Analyst do?

A Data Analyst collects, cleans, transforms and analyzes data to help businesses make better decisions. They create dashboards, reports, insights and forecasts using tools like Excel, SQL and Power BI.

---

2. What tools do Data Analysts use?

  • Excel (Advanced)
  • SQL (MySQL, PostgreSQL, SQL Server)
  • Power BI / Tableau
  • Python (Pandas)
  • Power Query
  • ETL Tools
---

3. How do you handle missing or inconsistent data?

Common techniques:

  • Remove nulls
  • Replace nulls (0, mean, median, mode)
  • Forward-fill / Back-fill
  • Correct formats (dates, numbers)
  • Identify duplicates

In Power Query: Replace Errors, Fill Down, Remove Rows In SQL: COALESCE(), CASE

---

4. What is the difference between a KPI and a Metric?

  • Metric → any measurable number (orders, revenue, calls received)
  • KPI → a metric that measures performance against goals (conversion rate, SLA %, sales growth)
---

5. Explain ETL.

ETL = Extract → Transform → Load Used for moving data from multiple sources into a structured format.

Example: Import Excel → Clean in Power Query → Load into Power BI.

---

6. What is a Star Schema?

A data modelling structure with:

  • One Fact Table (transactions)
  • Multiple Dimension Tables (Date, Customer, Product)

This improves Power BI performance & clean relationships.

---

7. How do you analyze trends in data?

Techniques include:

  • Moving Averages
  • YoY, MoM growth
  • Seasonality
  • Time-series charts
  • Forecasting
---

8. What is correlation?

Correlation shows the relationship between two variables (strength + direction).

+1 → strong positive -1 → strong negative 0 → no relationship

---

9. What is normalization?

Normalization organizes data in the database to reduce duplication and improve consistency.

Example: Separating customer table & orders table.

---

10. What does a typical Data Analyst project workflow look like?

  1. Understand business requirement
  2. Collect data
  3. Clean & transform data
  4. Analyze & visualize
  5. Generate insights
  6. Communicate results
---

11. What is the difference between Report and Dashboard?

  • Report → multiple pages, detailed tables, deep dive
  • Dashboard → one page, summary KPIs, visuals only
---

12. What is DAX?

DAX = Data Analysis Expressions Used to create dynamic calculations in Power BI. 

Total Sales = SUM(Sales[Amount])
---

13. What SQL queries do Data Analysts use most?

  • SELECT
  • JOIN
  • GROUP BY
  • WHERE
  • HAVING
  • WINDOW FUNCTIONS
  • CTEs
---

14. How do you validate data accuracy?

  • Check duplicate rows
  • Check missing values
  • Cross-verify totals
  • Check date formats
  • Compare SQL vs Power BI output
---

15. How do you explain insights to management?

Use simple language. Include recommendations. Focus on business impact.

Example: “Sales dropped 12% last month due to low stock availability in North region.”
---

16. What is A/B Testing?

A/B testing compares two versions of something to see which performs better.

---

17. What is a Cohort Analysis?

Cohort analysis groups customers/users based on their first interaction and tracks behavior over time.

---

18. Explain Outliers.

Outliers are values that are unusually high or low compared to the rest of the data. Often detected using Z-score or IQR.

---

19. What visualizations do Data Analysts commonly use?

  • Line chart
  • Bar/Column chart
  • Pie/Donut (limited use)
  • Heatmap
  • Boxplot
  • KPI cards
---

20. What skills should a Data Analyst have?

  • Excel (Advance)
  • SQL
  • Power BI/Tableau
  • Data Modeling
  • Data Cleaning
  • Communication
---
