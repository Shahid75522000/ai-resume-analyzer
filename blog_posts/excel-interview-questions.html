Excel Interview Questions & Answers (2026 Edition)

The most commonly asked Excel questions for MIS, Reporting, Analyst and Operations interviews.

Excel is still one of the most in-demand skills in 2026 for reporting, operations, MIS, and analyst jobs. These questions will help you prepare for real interviews quickly and confidently.

---

1. What is the difference between VLOOKUP and XLOOKUP?

  • VLOOKUP → Searches only to the right.
  • XLOOKUP → Searches left & right, supports exact match, and is more flexible.
Example: 
=XLOOKUP(A2, B2:B100, C2:C100)
---

2. What does IFERROR do?

IFERROR returns an alternative value when a formula produces an error. 

=IFERROR(VLOOKUP(A2, D:E, 2, FALSE), "Not Found")
---

3. What is the difference between COUNT, COUNTA, COUNTBLANK & COUNTIF?

  • COUNT → Counts numbers
  • COUNTA → Counts non-blank cells
  • COUNTBLANK → Counts empty cells
  • COUNTIF → Counts cells based on condition
---

4. What is a Pivot Table?

A Pivot Table summarizes large data using drag-and-drop fields.

Common use cases:
  • Sales by month
  • Calls by agent
  • Inventory by category
---

5. What are Absolute & Relative cell references?

Absolute reference: $A$1 (does not change when copied)
Relative reference: A1 (changes when copied)

---

6. What is Conditional Formatting?

Conditional formatting highlights cells based on rules (e.g., values > 1000, duplicates, top 10%).

---

7. What is the difference between CONCATENATE and TEXTJOIN?

  • CONCATENATE → Joins text but is limited
  • TEXTJOIN → Joins text with a delimiter, can ignore blank cells
=TEXTJOIN(", ", TRUE, A2:A10)
---

8. What is Power Query?

Power Query is used for data cleaning and automation:

  • Remove duplicates
  • Split columns
  • Merge tables
  • Change data types
M Language Example: 
= Table.RemoveDuplicates(#"Changed Type", {"ID"})
---

9. What is the difference between Remove Duplicates vs UNIQUE()?

  • Remove Duplicates → One-time action
  • UNIQUE() → Dynamic formula that updates automatically
---

10. What does FILTER() do?

FILTER returns only the rows that match the condition. 

=FILTER(A2:C100, C2:C100="Completed")
---

11. What are common Excel keyboard shortcuts?

  • Ctrl + Shift + L → Add Filter
  • Ctrl + T → Convert to Table
  • Ctrl + Arrow Keys → Move across data
  • Ctrl + Shift + Down/Right → Select data
---

12. How do you clean data in Excel?

  • TRIM
  • CLEAN
  • Find & Replace
  • Remove Duplicates
  • Text-to-columns
---

13. What is Data Validation?

Used to restrict inputs (e.g., dropdown lists, number ranges).

---

14. How do you protect a sheet or workbook?

Use Review → Protect Sheet / Protect Workbook. Often used in MIS reporting.

---

15. How do you find unique values?

Use UNIQUE() or Pivot Table → Rows.

---

16. What is Excel Table and why use it?

Excel Table helps with structured references and automatic range updates.

Example: 
=SUM(Table1[Sales])
---

17. Explain XLOOKUP with multiple conditions.

Use TEXTJOIN inside XLOOKUP: 

=XLOOKUP(A2&B2, D:D&E:E, F:F)
---

18. How do you remove spaces in Excel?

=TRIM(A2)

---

19. How do you calculate percentage?

=Actual / Target then format as %

---

20. How to combine first and last name?

=A2 & " " & B2
